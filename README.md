# Product Inventory API

This is a **beginner-friendly Node.js + Express API** built as a group capstone project.
The API simulates a simple **product inventory system** where products can be created,
viewed, updated, and deleted.

This project is focused on **learning backend fundamentals**, Git collaboration, and API testing.
